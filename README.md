# Manus Skill: System Indexer

Ein Manus-Skill, der den zentralen Wissensgraphen des Manus-Systems verwaltet.

## Installation in Manus

Importiere diesen Skill über: `https://github.com/Friseurehattstedt/manus-skill-system-indexer`

## Was dieser Skill tut

Er gibt Manus Zugriff auf den vollständigen Wissensgraphen aller Skills und Konnektoren — als persistentes, maschinenlesbares Gedächtnis.

## Wissensgraph aktualisieren

```bash
python3 scripts/build_graph.py
```
